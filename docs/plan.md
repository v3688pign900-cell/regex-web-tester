# Plan

## Goal
Build a static Regex testing tool that runs on GitHub Pages and also works offline by opening `index.html` directly.

## Core requirements
- HTML, CSS, JavaScript only
- No external libraries, CDN, or network APIs
- Immediate local regex testing with safe error handling
- Responsive UI for phone, tablet, and desktop
- Match summary, match index, matched groups, and highlight output
- Result copy and export support

## Deliverables
- `index.html`
- `style.css`
- `app.js`
- `README.md`
- `.gitignore`
- `docs/design.md`
- `docs/security_design.md`
- `docs/user_guide.md`
